```git clone https://github.com/whitneygriffith/Simple-React-Express-App```

```cd Simple-React-Express-App```

#### Start Backend

```npm install```

```node server.js``` this will start the backend server

#### Start Frontend

```cd client```

```npm install```

```npm start``` this will start the frontend-react app
